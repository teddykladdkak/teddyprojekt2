Dokument Genererare









Instruktioner för att skapa dokument

Excel-filens roll

Excel-filen innehåller den data som används för att skapa dokumenten i Word.

  • Kolumnrubriker: Den första raden i Excel-filen innehåller rubriker för varje kolumn. Dessa används som referenser i Word-mallen.
    • Undvik specialtecken som å, ä, ö samt mellanslag i rubrikerna.
    • Rubrikerna syns inte i de färdiga dokumenten.
  • Rader: Varje rad (förutom rubrikraden) motsvarar ett dokument som ska skapas.
    • Exempel: Om Excel-filen innehåller fyra rader med data utöver rubrikraden, genereras fyra dokument.

Word-mallens roll

Word-mallen är en .docx-fil som fungerar som mall för de färdiga dokumenten.

  • Referenser i texten: För att hämta data från Excel till Word används formatet "{rubriknamn}", där rubriknamn motsvarar en kolumnrubrik i Excel.
    • Exempel: Om Excel-filen har en kolumnrubrik som heter "verksamhetens_namn", skriver du "{verksamhetens_namn}" på den plats i Word-mallen där informationen ska infogas.
  • Fel "undefined": Om referens i wordfilen inte hittas som rubrik i Excel ersätts den med "undefined" istället.

Filnamngivning

De genererade Word-filerna behöver unika filnamn.

  • Ange vilken kolumn i Excel som ska användas som referens för filnamnen genom att skriva kolumnens rubrik i textrutan.
    • Exempel: Om du vill använda kolumnen "verksamhetens_namn" för filnamnen, skriver du detta i textrutan.

Wordfiler till PDF

För att konvertera wordfilerna till PDF kan du använda "Convert Doc to PDF" (Gratis programvara).

Så här använder du tillägget:

  1. Klicka på programvaruikonen.
  2. Välj word-filerna som du vill konvertera.
  3. Tryck på "Konvertera" och vänta på att PDF-filen skapas.

Observera att detta är en offline-programvara, ingen internetanslutning krävs för att konverteringen ska fungera.
